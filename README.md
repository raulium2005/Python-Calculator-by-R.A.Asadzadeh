To start calculating, type the number corresponding to the operation:
  1- addition
  2- subtraction
  3- multiplication
  4- division
  5- exponentitation

Next, you write two numbers you want

NOTE: the calculations will be done like this:      (a- first number  b- second number)
  addition:       a+b
  subtraction     a-b
  multiplication  a*b
  division        a/b
  exponentitation a^b
